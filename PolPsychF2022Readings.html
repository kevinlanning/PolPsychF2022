2022-08-20

The current set of references is arranged in a series of modules below. Note that modules may extend across several classes.

There are eleven numbered modules, which we will cover in the first ten weeks of the class. Following this, we will select topics for the remaining classes from modules A-H. Students will be expected to help lead these meetings, which will also serve as the basis for their term papers.

All of these papers may be found in the files section in Canvas.
